# Timeline publique — École de guerre

Version publique en lecture seule de la frise chronologique de l'École de guerre (Guerre froide, 1943–1975).

## Fonctionnalités

- Visualisation chronologique des événements, périodes et mandats politiques
- Filtres par type, catégorie, thème et période
- Recherche plein texte (titre, acteurs, lieux)
- Panneau de détail par événement
- Interface responsive : desktop, tablette et mobile

## Utilisation mobile

Sur écran mobile (≤ 768 px) :

- Le panneau de filtres s'ouvre via le bouton **☰ Filtres** et se ferme avec **Valider**
- Le contenu est affiché en colonne unique chronologique
- Le header reste visible en permanence lors du défilement
